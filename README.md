# 2025西工大通信工程专业综合实验
2025西工大通信工程专业综合实验，仅供参考，切勿抄袭


本次通信工程专业综合实验通过GNU Radio软件无线电平台，系统性地完成了AM、FM与MPSK三类调制解调技术的仿真研究，成功将理论知识与工程实践相结合，深化了对通信系统核心原理的理解。

在AM调制实验中，重点探究了采样定理对信号保真的根本性约束。通过改变系统采样率与载波频率，直观观测并分析了频谱混叠现象及其对音质的破坏性影响，明确了载波频率必须高于基带信号最高频率以避免信息丢失的原则。同时，实验揭示了接收端低通滤波器在滤除高频分量、还原基带信号中的关键作用。

FM调制实验则聚焦于其优越的抗噪声性能。通过对比相同噪声环境下AM与FM的解调输出，验证了FM在信噪比上的显著优势。此外，通过调整预加重时间常数，深入理解了预/去加重技术如何通过提升高频分量来抑制解调后高频噪声，从而优化整体收听质量。

MPSK调制实验构成了数字通信系统的核心探索。从最简单的BPSK直接解调出发，逐步引入并验证了线性均衡（对抗码间串扰）、定时同步（Polyphase Clock Sync）与载波同步（Costas Loop）在恢复稳定星座图中的不可或缺性。实验揭示了采样率匹配、同步环路参数（如Taps数）设定与发射滤波器相匹配的重要性，并成功通过差分编解码技术解决了载波同步中固有的相位模糊（倒π现象）。最终，通过将星座图修改为QPSK并配套调整系统参数，实现了从BPSK到更高阶调制的迁移。

综上，通过本系列实验，不仅巩固了幅度、频率及相位调制的基础理论，更通过动手仿真，掌握了构建一个完整的数字接收机所必需的功能模块、连接顺序及参数整定原则，锻炼了我利用仿真工具分析并解决通信系统中同步、均衡、抗噪声等核心工程问题的能力。
